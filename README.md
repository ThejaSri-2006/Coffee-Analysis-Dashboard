# ☕ Coffee Sales Analysis Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes coffee sales data to identify top-selling coffee varieties, revenue trends, and customer purchasing patterns.

## 📊 Tools Used
- Power BI
- CSV Dataset
- DAX
- Power Query

## 📂 Files Included
- Coffee.pbix
- coffee_data.csv
- Coffee product images
- Background image

## 📈 Dashboard Insights
- Top-selling coffee products
- Sales trends
- Revenue analysis
- Product-wise performance

## 🚀 How to Use
1. Download the repository.
2. Open `Coffee.pbix` in Power BI Desktop.
3. Refresh the data if needed.
